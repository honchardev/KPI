API methods

  • [ /api ] or [ /api/help ]
     Here you can find help if you are stuck or simply need navigation help.
  • [ /api/directors ]
     Here you can see all the information about all the directors in xml format.
  • [ /api/directors/about ]
     Here you can seel information about directors.xml file.
  • [ /api/directors/{id} ] where {id} describes certain director id number.
     Here you can see information about certain director, depending on his id number in xml format.
     For example, check out this link - link to director with id=1.

    HTML

  • [ /home ] or simply [ / ]
     Home page with lot of information on it.
  • [ /help ] or [ /home/help ]
     Help page where you can find help if you are stuck or simply need navigation help.
  • [ /directors ]
     Page where you can find information about all directors.
  • [ /directors/{id} ] where {id} describes certain director id number.
     Here you can find information about director with certain number.
     If there are no directors with id you are looking for, you`ll receive an empty page.
     For example, check out this link - link to director with id=1.
  • [ /new-director ]
     Page where you can insert a new director.
